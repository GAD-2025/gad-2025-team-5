# Frontend 기능 상세 설명서

이 문서는 React로 구현된 Frontend 애플리케이션의 각 페이지와 핵심 기능들을 상세하게 설명합니다.

## 📄 페이지 및 기능

### 1. 스플래시 및 인증

-   **`SplashScreen.jsx`**
    -   **역할**: 애플리케이션 로딩 시 가장 먼저 사용자에게 노출되는 화면입니다.
    -   **기능**:
        -   "책담"이라는 타이틀을 잠시 보여줍니다.
        -   3초 후에 자동으로 `/login` 경로로 사용자를 이동시킵니다.

-   **`LoginPage.jsx`**
    -   **역할**: 사용자 로그인을 처리하는 페이지입니다.
    -   **기능**:
        -   **로그인**: 이메일과 비밀번호를 입력받아 로그인을 시도합니다.
            -   로그인 성공 시, 신규 사용자는 온보딩 페이지 (`/onboarding`)로, 기존 사용자는 메인 홈 페이지 (`/home`)로 이동합니다.
            -   실패 시, 에러 메시지를 담은 팝업을 표시합니다.
        -   **회원가입 이동**: `회원가입` 버튼 클릭 시, 회원가입 페이지 (`/signup`)로 이동합니다.
        -   **비밀번호 찾기**: `비밀번호를 잊으셨나요?` 링크를 제공합니다 (현재는 콘솔 로그만 출력).

-   **`RegisterPage.jsx`**
    -   **역할**: 신규 사용자 등록 절차의 첫 단계를 담당합니다.
    -   **기능**:
        -   **정보 입력**: 프로필 사진(선택), 닉네임, 이메일, 비밀번호를 입력받습니다.
        -   **닉네임 중복 확인**: `중복 확인` 버튼을 통해 서버에 동일한 닉네임이 있는지 비동기적으로 확인합니다. 사용 가능한 닉네임이어야만 가입이 가능합니다.
        -   **약관 동의**: 서비스 이용약관, 개인정보 수집 동의, 만 14세 이상 확인을 위한 체크박스를 제공하며, 모두 동의해야 합니다.
        -   **가입하기**: `가입하기` 버튼 클릭 시, 모든 정보가 유효하면 입력된 정보를 로컬 스토리지에 임시 저장하고 다음 단계인 온보딩 페이지 (`/onboarding`)로 이동합니다.

-   **`Onboarding.js`**
    -   **역할**: 신규 가입자의 취향을 파악하기 위한 2단계 온보딩 프로세스입니다.
    -   **기능**:
        -   **1단계 (장르 선택)**: 사용자에게 여러 장르를 제시하고, 최소 하나 이상의 관심 장르를 선택하게 합니다.
        -   **2단계 (도서 선택)**: 베스트셀러 목록을 보여주고, 이 중에서 좋아하는 책을 3권 이상 선택하게 합니다. 검색 기능으로 원하는 책을 찾을 수도 있습니다.
        -   **최종 가입**: `완료` 버튼 클릭 시, `RegisterPage.jsx`에서 임시 저장된 사용자 정보와 온보딩에서 선택된 취향(장르, 도서) 데이터를 모두 서버로 전송하여 최종 회원가입을 완료합니다. 성공 시, 사용자는 홈 페이지 (`/home`)로 이동합니다.

### 2. 메인 화면 및 도서 목록

-   **`Home.js`**
    -   **역할**: 앱의 핵심 콘텐츠를 보여주는 메인 대시보드입니다.
    -   **기능**:
        -   **헤더**: 로고, 검색, 알림, 전체 카테고리 메뉴 버튼이 포함되어 있습니다.
        -   **사용자 맞춤 배너**: 로그인한 사용자에게 맞춤 인사말과 추천 도서 정보를 담은 배너를 보여줍니다.
        -   **추천 도서**: '오늘의 추천', '인기', '내 취향 맞춤' 탭으로 구분된 추천 도서 목록을 가로 스크롤 형태로 제공합니다.
        -   **실시간 매물 도서**: '새 매물', '가격 인하' 탭으로 나뉜 실시간 등록 도서 목록을 보여줍니다.
        -   **커뮤니티 인기글**: 커뮤니티의 인기 게시글 일부를 보여주며, 클릭 시 해당 게시글 상세 페이지로 이동합니다.
        -   `전체 보기`/`더보기` 링크를 통해 각 목록의 전체 페이지(`List.js` 또는 `CommunityPage.jsx`)로 이동할 수 있습니다.

-   **`CategoryPage.jsx`**
    -   **역할**: 특정 장르에 해당하는 도서들만 모아서 보여주는 페이지입니다.
    -   **기능**:
        -   Home 화면의 전체 카테고리 메뉴에서 특정 하위 카테고리 클릭 시, 해당 카테고리 이름과 함께 관련 도서 목록을 표시합니다.

-   **`List.js`**
    -   **역할**: 다양한 종류의 도서 목록을 표시하는 범용 리스트 페이지입니다.
    -   **기능**:
        -   Home 화면에서 '추천 도서' 또는 '실시간 매물 도서' 섹션의 `전체 보기`를 클릭했을 때 해당 전체 목록을 보여줍니다.

-   **`BookDetail.jsx`**
    -   **역할**: 한 권의 책에 대한 모든 상세 정보를 제공합니다.
    -   **기능**:
        -   **상세 정보**: 책 이미지, 제목, 등록일, 가격, 판매자 정보(신뢰도 포함), 판매자의 한줄평, 책 상태 설명, 관련 분류(장르) 등을 표시합니다.
        -   **좋아요 (관심 등록)**: 하트 아이콘을 클릭하여 해당 책을 관심 목록에 추가하거나 삭제할 수 있습니다. (로그인 필요)
        -   **채팅하기**: 하단의 `채팅하기` CTA 버튼을 통해 판매자와 1:1 채팅을 시작할 수 있습니다.
        -   **구매하기**: `구매하기` CTA 버튼을 통해 해당 도서의 주문/결제 페이지 (`/payment`)로 이동합니다.

### 3. 도서 등록

-   **`Register.js`**
    -   **역할**: 판매할 중고 도서를 등록하는 절차의 첫 페이지입니다.
    -   **기능**:
        -   **책 정보 인식**: 바코드 스캔(카메라 이용) 또는 ISBN 번호 수동 입력을 통해 책 정보를 자동으로 불러올 수 있습니다.
        -   **정보 입력**: 사진 등록(최대 5장), 책 제목, 작가, 장르, 책 설명, 한줄 소감, 판매 가격을 직접 입력받습니다.
        -   **옵션 설정**: '가격 제안 받기' 허용 여부와 '배송 옵션'(무료배송/택배비 별도)을 선택할 수 있습니다.
        -   `다음` 버튼 클릭 시, 입력된 모든 정보를 컨텍스트에 저장하고 다음 등록 페이지 (`/register2`)로 이동합니다.

-   **`RegisterPage2.js`**
    -   **역할**: 도서 등록의 마지막 단계로, 책의 상태를 결정합니다.
    -   **기능**:
        -   **상태 등급 선택**: S급부터 D급까지, 책의 상태를 나타내는 등급 중 하나를 선택합니다.
        -   **등록하기**: `등록하기` 버튼 클릭 시, 이전 페이지에서 입력된 정보와 상태 등급을 합쳐 서버로 전송합니다.
        -   등록이 진행되는 동안 로딩 페이지 (`/registering`)를 잠시 보여줍니다.
        -   성공 시, 방금 등록된 책의 상세 페이지 (`/books/:id`)로 이동합니다.

-   **`RegisteringPage.jsx`**
    -   **역할**: 도서 등록 요청이 서버에서 처리되는 동안 사용자에게 보여지는 로딩 화면입니다.
    -   **기능**: "등록 중이에요!" 라는 메시지를 표시하여 사용자에게 대기 상태임을 알립니다.

### 4. 구매 및 결제

-   **`PaymentPage.js`**
    -   **역할**: 선택한 도서를 구매하기 위한 주문/결제 페이지입니다.
    -   **기능**:
        -   **주문 상품 확인**: 구매할 책의 정보(이미지, 제목, 가격 등)를 보여줍니다.
        -   **배송지 정보**: 받는 사람, 주소, 연락처 정보를 표시합니다. (현재는 고정값)
        -   **결제 수단**: 신용카드, 무통장입금 등 다양한 결제 수단을 선택할 수 있는 UI를 제공합니다. (현재는 UI만 구현)
        -   **최종 결제 금액**: 상품 금액과 배송비를 합산한 총 결제 금액을 명확히 보여줍니다.
        -   `결제하기` 버튼 클릭 시, 결제가 완료되었다는 알림을 띄우고 이전 페이지로 돌아갑니다.

### 5. 커뮤니티 및 채팅

-   **`CommunityPage.jsx`**
    -   **역할**: 사용자들이 자유롭게 소통하는 커뮤니티 공간입니다.
    -   **기능**:
        -   **게시글 목록**: 작성된 게시글들을 리스트 형태로 보여줍니다.
        -   **필터링**: '전체'와 '인기글' 탭을 통해 게시글 목록을 필터링할 수 있습니다.
        -   **글 작성**: 우측 하단의 플로팅 버튼(+)을 통해 새 게시글을 작성하는 페이지로 이동할 수 있습니다. (현재 작성 페이지는 미구현)

-   **`CommunityPostDetail.jsx`**
    -   **역할**: 커뮤니티 게시글 하나에 대한 상세 내용을 보여줍니다.
    -   **기능**:
        -   **내용 확인**: 게시글의 전체 텍스트, 이미지, 작성자 정보, 작성 시간을 볼 수 있습니다.
        -   **상호작용**: 게시글에 '좋아요'를 누르거나 해제할 수 있습니다.
        -   **댓글**: 하단에 댓글을 작성하고 등록할 수 있으며, 다른 사용자들이 남긴 댓글 목록을 볼 수 있습니다.

-   **`ConversationListPage.jsx`**
    -   **역할**: 사용자의 모든 채팅방 목록을 보여주는 페이지입니다.
    -   **기능**:
        -   각 채팅방 별로 상대방 닉네임, 관련 도서 제목, 마지막 메시지, 수신 시간, 읽지 않은 메시지 수를 표시합니다.
        -   채팅방 클릭 시, 해당 1:1 채팅 화면(`ChatPage.jsx`)으로 진입합니다.

-   **`ChatPage.jsx`**
    -   **역할**: 판매자와 구매자 간의 1:1 대화가 이루어지는 실제 채팅 화면입니다.
    -   **기능**:
        -   **상품 정보**: 채팅방 상단에 대화와 관련된 도서 정보를 표시하여 쉽게 확인할 수 있습니다.
        -   **메시지 전송**: 하단의 입력창을 통해 텍스트 메시지를 보내고 실시간으로 대화할 수 있습니다.
        -   **자동 응답 (데모용)**: 사용자가 특정 키워드(예: '구매 가능', '직거래')를 포함한 메시지를 보내면, 판매자 성향(친절, 사무적 등)에 맞춰 미리 정의된 답변이 자동으로 돌아오는 기능이 구현되어 있습니다.

### 6. 마이페이지

-   **`MyPage.jsx`**
    -   **역할**: 사용자 개인의 활동 내역과 정보를 종합적으로 보여주는 페이지입니다.
    -   **기능**:
        -   **프로필 정보**: 사용자의 프로필 사진, 닉네임, 이메일을 표시합니다.
        -   **활동 내역**: '판매 내역', '구매 내역', '관심 목록'으로 이동할 수 있는 메뉴를 제공합니다.
            -   '판매 내역' 섹션에는 사용자가 등록한 도서 목록이 표시됩니다.
        -   **로그아웃**: `로그아웃` 버튼을 클릭하여 현재 계정에서 로그아웃하고 초기 화면으로 돌아갈 수 있습니다.
